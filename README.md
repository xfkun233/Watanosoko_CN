### Point

### 概要

在流传着『异形现身于海、宣告灾厄降临』这一传说之地，发生了一连串离奇案件。

大学社团「不可思议研究会」的成员九段坂山彦，收到了一位在联合合宿中相识的少女——雨野有绘带来的奇妙消息。

据说，在她熟人正策划“振兴城镇”计划的那片土地的海岸上，漂来了一个非人非鱼的“某种东西”。

山彦因为被认为精通怪谈奇闻，于是被强行拉去随行。

尽管他们抵达了那片流传着「异形现身于海、宣告灾厄降临」传说之地，但那个叫他们来的人却下落不明。

他们与负责城镇振兴的成员汇合后开始搜寻其下落，然而——

最终发现的尸体，竟然没有头颅。
